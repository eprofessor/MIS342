Overview

You can work together and help each other on this project, but each student must hand in their own work.

The aim of this part is to establish a connection between a MySQL and Access database.
This project illustrates the use of Microsoft Access as a front end for another database server.

Note: For online help with MySQL 5.x see: http://dev.mysql.com/doc/refman/5.7/en/index.html

  1. Locate the folder 'P2DataAndScripts' on the Virtual Machine desktop, double click to open it up.
