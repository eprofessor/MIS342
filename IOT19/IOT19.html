Learning Objectives

These instructions will show you how a Raspberry Pi and GrovePi sensors can retrieve data from the physical world.
You send this sensor data to a publicy accessible cloud provider.

More information on the Adafruit IO system that you will use is available online.
These systems allow data collection in the cloud from devices connected to your Raspberry Pi-such as peripherals and connected sensors that measure light level, sound level, temperature level, humidity level and more.

After completing this assignment you will be able to:

  1. Investigate sensors
  2. Run Python programs on a Raspberry Pi that communicate with sensors and send data to the cloud
  3. Understand use of removable USB Flash Drives to store confidential information needed in your Python programs, like passwords and tokens.
  4. Discuss uses of Raspberry Pi/Grove Pi devices to solve problems and address opportunities

Exercise 1-Adafruit IO Account
  1. These steps are performed on your computer.
  2. Open a browser.
  3. Create an Adafruit.io account by logging into Adafruit IO and following the directions.
  4. Using your Adafruit account, obtain a Username and Key to connect to your Data Feed

    Obtaining Adafruit Username and Key

  5. Insert a USB Flash Drive into your computer.
  6. Open Notepad and create a text file named 'config.txt'
  7. On the first line type in your Adafruit account 'Username'
  8. Copy your Adafruit account 'Key' and paste it on the second line
  9. Save the file as 'config.txt' to the root of the USB Flash Drive
  10. Using Windows File Explorer right click the USB Flash Drive and rename it:  'MYDATA'

    Renaming USB Flash Drive

  11. Close File Explorer, eject the USB Flash Drive, and then remove it from your computer.
  12. Insert the USB Flash Drive, named 'MYDATA' and containing the file 'config.txt' into the Raspberry Pi.
  13. Use VNC to transfer the file 'adafruit.py' from your computer to the Raspberry Pi, and place it in the folder '/home/pi/Desktop/IoT_Files'
    Note- a copy can be found in the OneDrive - MNSCU/ReadOnly/IoT/IoT_PythonFiles folder.
Exercise 2-Send Sensor Data to the Cloud
  1. These steps are performed on your Raspberry Pi.
  2. Launch a Terminal window
  3. Run the following command to ensure the Python Package installer is present.
    sudo apt-get install python3-pip
  4. Run the following command to ensure that the Networkscan library is installed.
    pip3 install adafruit-io
  5. Launch Thonny
  6. Load the program /home/pi/Desktop/IoT_Files/adafruit.py
  7. Run the program adafruit.py>
  8. Examine the output shell to make sure that temperature and humidity data appears.
  9. Review the program adafruit.py and think about improvements that could be made.
