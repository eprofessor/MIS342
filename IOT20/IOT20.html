Learning Objectives

These instructions will show you how to use Microsoft Access to link to a MySQL/Maria database server on a Raspberry Pi.
This provides the foundation for collecting data from sensors and programs on a Raspberry Pi, storing the data in a MySQL database on the Raspberry Pi, then viewing the data remotely across a network.

More information on MySQL is available online.

After completing this assignment you will be able to:

  1. Investigate ODBC for Access to MySQL
  2. Run MySQL
  3. Use Microsoft Access on a network to view data stored on a MySQL database on a Raspberry Pi

Exercise 1-Start MySQL

MySQL/MariaDB is already installed on the Raspberry Pi, see IOT10.

  1. These steps are performed on the Raspberry Pi
  2. Start VNC to retrieve your Raspberry Pi IP address.
    In the next exercise you will need to enter it into the TCP/IP field to establish a connection from your computer to the Raspberry Pi.

    Rasbberry Pi TCP/IP Server Address

  3. Open a Terminal Window
  4. Log into the MySQL installation, the default password is blank.
    In the terminal window type:
    sudo mysql -u root -p -h localhost
    Press Enter when prompted for the password.
    Type status to see MySQL settings.
    MySQL status
  5. Note: Once MySQL is running, to verify it is working, in a Terminal Window type:
    sudo /etc/init.d/mysql status
    MySQL running successfully
  6. Once MySQL is running, verify that the world database is installed and tables with data are available by entering the following in the MySQL command line/Terminal Window:
    use world

    MySQL Using World Database

  7. Run the show tables command, you should see three tables:
    MariaDB [world]> show tables;
    +-----------------+
    | Tables_in_world |
    +-----------------+
    | city |
    | country |
    | countrylanguage |
    +-----------------+
    3 rows in set (0.001 sec)
  8. Experiment with commands you learned in Project 3 to retrieve data from these tables:
    SELECT COUNT(*) FROM city;
    SELECT * FROM country;
  9. Congratulatons! Your Raspberry Pi is running the MySQL/Maria database server correctly.
    With a little more work, you can use this database to collect information from sensors attached to your Raspberry Pi.
Exercise 2-Connect With ODBC

These steps are performed on your computer.

WSU laptops use 64-bit Windows with 64-bit Microsoft Access.
You must install and use the 64-bit ODBC MySQL connector
See MIS342 Project 2, Part 1 for details.

In this exercise learn to connect to the Raspberry Pi MySQL database from Microsoft Access on another computer over a network.

  1. Log into D2L.
    From the Course Info widget click the 'OneDrive ReadOnly Files' link and browse to:
    OneDrive - MNSCU/ReadOnly/IoT/ODBC_Installer_MySQL folder.
    There you will find the 64-bit MySQL ODBC .msi file.
    Download it to your computer.

    Retrieve ODBC Connector

    Note:  for the most recent version of the .msi package browse to this url, install the appropriate ODBC version for MySQL , making sure to follow the instructions carefully:
    https://dev.mysql.com/doc/connector-odbc/en/connector-odbc-installation-binary-windows.html 

    ODBC Installer

    MySQL Installer
    Note that you do not have to set up an account, just download and run the installer.

  2. Locate the .msi file, double click to begin the installation process and do a 'Typical' install.

    Choose Typical Install


    Make sure to use the 64-bit version of ODBC and the 64-bit MySQL connector
  3. Once installed, click Start, type in 'ODBC', find the 64-bit version and click to start.
  4. Under the System DSN tab, Add a new Data Source, using the MySQL ODBC 8 Unicode driver.
    note: ANSI driver does not work on all systems, but Unicode seems to

    ODBC Installer

  5. Set the following parameters to establish your connection to the world database on the Raspberry Pi
    The TCP/IP Server value will be different, and was retrieved in the previous Exercise.
    Username: user
    Password: user
    Database: world

    ODBC Setup

  6. Troubleshoot as needed, and once you have demonstrated a successful connection click 'OK' to dismiss the dialog boxes.
  7. Create a new Microsoft Access Database, name it 'world' and save it as "world.accdb" in your ' ' folder.
  8. From the menu choose External Data>New Data Source> From Other Sources>ODBC Database: and complete the steps to setup the connection to the World database on the Raspberry Pi:
    Access ODBC connection
  9. In the navigation pane you will see the three tables from the world database: city, country and country language.
    Access linked to MySQL on Raspberry Pi!
  10. Use Acccess to add a record to the city table.
    Do not specify an ID value.
    Name: Winona
    CountryCode: USA
    District: Minnesota
    Population: 25000
    Take a screen shot of it showing this new record in the Raspberry Pi MySQL world database
